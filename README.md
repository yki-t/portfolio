# ゆっくり解説ジェネレータ(仮)

## これは何？

ゆっくり解説をプログラムから動的に作成するプログラム。

各場面ごとの
- スクリプト (セリフ)
- BGM・背景動画
- 表示画像
- 表示キャラクタ (表情もいる？)

等を設定ファイルに書き込んでビルドすると、動画ができるような仕組み


## やること

- そんなことできるのか？を検証
- できそうならライブラリ選定
- 時期を決める
    - 早めにプロトタイプを作って
    - テスト動画を作成して
    - 機能追加と並行して一般公開
